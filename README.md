# Generation Alpha Awareness & Intervention Platform

A production-grade, real-time awareness and early-intervention web platform for children with learning disabilities (LD).

## 🚀 Vision
To educate parents, educators, and policy makers on the psychological burden of digital learning on Generation Alpha and provide validated psychometric screening tools adapted for digital environments.

## 🛠 Tech Stack
- **Frontend**: Next.js 14 (App Router), TypeScript, Tailwind CSS, Framer Motion, lucide-react.
- **Backend**: NextAuth.js v5, Prisma ORM, PostgreSQL.
- **AI**: Claude 3.5 Sonnet Integration (Psychology & Crisis Detection).

## 📁 Key Modules
- **Public Awareness Hub**: Interactive scrollytelling and statistics on digital learning pressure.
- **Screening Portal**: Validated psychometric tools (SDQ, SCARED, DSIC) with adaptive forms.
- **Child Safe Zone**: A trauma-informed, illustrated space for mood logging and grounding exercises.
- **Educator Dashboard**: Real-time classroom wellness analytics and student priority flags.
- **Resource Library**: Clinically reviewed articles and toolkits for LD support.
- **Crisis Support Layer**: Global helpline integration and AI-driven triage.

## 🏃 Running Locally
1. Clone the repository.
2. Install dependencies: `npm install`.
3. Set up your environment variables in `.env` (DATABASE_URL, NEXTAUTH_SECRET, etc.).
4. Run Prisma migrations: `npx prisma migrate dev`.
5. Start the dev server: `npm run dev`.

## 🧠 Clinical Framework
Aligned with DSM-5-TR, Polyvagal Theory, ACEs Framework, and Cognitive Load Theory. Designed with Trauma-Informed principles (Safety, Transparency, Empowerment).
